Each example below is a **full script you can copy & run**.

- **Simple primitives**: place basic primitives and render to SVG
- **Random primitives**: reproducible random scene via a fixed seed
- **Conic section**: render the intersection curve of Plane × Cone

## Interactive Examples

Each example is an **interactive viewer** with drag/wheel orbit controls on its own page.
