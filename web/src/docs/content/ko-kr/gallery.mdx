아래 예제들은 **문서에서 바로 복사해서 실행할 수 있는 전체 스크립트**로 제공됩니다.

- **단순 primitives**: 기본 도형들을 배치하고 SVG로 렌더링
- **랜덤 도형 생성**: seed 기반으로 재현 가능한 랜덤 씬 생성
- **conic section**: Plane × Cone 교차 곡선(원뿔 곡선) 렌더링
